# jacobsalway.com

Repository for my personal website at [jacobsalway.com](https://jacobsalway.com). Inspired by https://github.com/flexdinesh/blogster

- **Language**: Typescript
- **Frameworks**: Astro
- **Styling**: Tailwind CSS
- **Hosting:** Vercel

## Running locally

```
git clone https://github.com/jacobsalway/jacobsalway.com
cd jacobsalway.com
pnpm install
pnpm dev
```
