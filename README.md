# jacobsalway.com

Repository for my personal website at [jacobsalway.com](https://jacobsalway.com)

- **Language**: Typescript
- **Frameworks**: [Astro](https://astro.build)
- **Styling**: Tailwind CSS
- **Hosting:** Vercel

## Running locally

```
git clone https://github.com/jacobsalway/jacobsalway.com
cd jacobsalway.com
pnpm install
pnpm dev
```
